# 🌟 North Star Invest

AI-powered financial intelligence platform for unified multi-asset investing and macro-aware decision-making.

## Problem
Retail investors use multiple platforms for stocks, mutual funds, gold, bonds, REITs, and alternatives, leading to fragmented portfolios and poor financial awareness.

## Solution
A unified AI-driven financial intelligence system with:
- AI Financial Copilot
- Macro Intelligence Engine
- Multi-asset portfolio dashboard
- Scenario-based financial simulations

## Tech Stack
Frontend: Next.js + Tailwind + Recharts  
Backend: FastAPI  
Database: PostgreSQL  
AI: GPT-4o via LangChain  
Market Data: Alpha Vantage / Finnhub / Yahoo Finance  

## Vision
To build an AI-native financial operating system that helps users understand, plan, and optimize their entire financial life.
