# 🌟 North Star Invest

> AI-powered financial intelligence platform for unified multi-asset investing and macro-aware decision-making.

---

## 🚨 Problem

Retail investors today face fragmented investing across multiple platforms:
- Stocks
- Mutual Funds
- Gold
- Bonds
- REITs / InvITs
- Alternative assets

This leads to:
- No unified portfolio view
- Poor diversification decisions
- Limited understanding of macroeconomic impact
- Emotional and reactive investing behavior

---

## 💡 Solution

North Star Invest is an AI-native financial intelligence platform that unifies multi-asset investing with contextual decision support.

It goes beyond aggregation by helping users:
- Understand their portfolio holistically
- Interpret macroeconomic conditions
- Make informed, goal-based financial decisions

---

## 🧠 Core Features

- AI Financial Copilot for personalized insights  
- Macro Intelligence Engine (inflation, interest rates, liquidity impact)  
- Unified multi-asset portfolio dashboard  
- Risk & diversification analysis  
- Scenario-based “what-if” simulations  
- Financial awareness and education layer  

---

## 🏗️ System Architecture

Frontend → Backend → Data Layer → AI Layer → Insights → Visualization

- Frontend: Next.js + Tailwind + Recharts  
- Backend: FastAPI (Python)  
- Database: PostgreSQL  
- AI: GPT-4o via LangChain  
- Market Data: Alpha Vantage / Finnhub / Yahoo Finance  

See detailed architecture in `/docs/architecture.md`

---

## 🔁 Process Flow

1. User onboarding (goals, risk, income, portfolio)
2. Data aggregation (portfolio + market + macro data)
3. AI processing (Copilot + Macro Engine in parallel)
4. Visualization (interactive dashboards)
5. Feedback loop (continuous personalization)

---

## 📊 Key Differentiators

- Multi-asset unification including alternative investments  
- Macro-aware AI decision intelligence  
- Behavioral finance insights  
- Portfolio-first rather than product-first approach  
- Simulation-driven financial planning
